## Quick Reference

### Tech Stack

```
Frontend: Next.js 14 + TypeScript + Zustand + TailwindCSS
Backend:  NestJS 10 + Prisma ORM + PostgreSQL
Realtime: Socket.io + Server-Sent Events
```

### Common Workflows

- **New API endpoint**: `api-developer` → `database-manager`
- **New UI feature**: `frontend-expert` → `state-management-expert`
- **Realtime feature**: `realtime-communication-expert`
- **Complex feature**: `complex-feature-implementation`
