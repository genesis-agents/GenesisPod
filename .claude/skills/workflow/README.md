## Quick Reference

### Workflow Cycle

```
Plan → Execute → Verify → Complete
  ↑                        ↓
  └────── Iterate ←────────┘
```

### Planning Process

1. Gather requirements
2. Identify affected files
3. Break into atomic tasks
4. Define success criteria
5. Execute with verification

### Verification Checklist

- [ ] Type check passes
- [ ] Tests pass
- [ ] Lint clean
- [ ] Feature works as expected
- [ ] No regressions
