## Quick Reference

### Data Flow Architecture

```
Sources → Collectors → Processing → Storage → Knowledge Graph → RAG
```

### Key Components

- **Collectors**: Web crawlers, RSS feeds, API integrations
- **Processing**: Deduplication, quality scoring, embedding
- **Knowledge Graph**: Entity extraction, relationship mapping
- **RAG**: Context retrieval for AI responses
