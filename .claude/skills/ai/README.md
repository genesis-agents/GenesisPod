### Architecture Layers

```
AI Engine (Core) → AI Teams (Collaboration) → AI Apps (User-facing)
```

### Common Entry Points

- **New AI feature**: Start with `ai-architecture-layering` to decide placement
- **LLM integration**: Use `ai-service-expert` for provider setup
- **Multi-agent**: Use `ai-teams-expert` for collaboration patterns
- **Content generation**: Use `document-generation` or `writing-quality`
