01核心论点 · 2028–2030 算力底座的本质转变

压测反复揭示一个系统性偏误：8 个维度各自把"需求侧持续高增长"当外生给定，并把同一组下行因子（scaling 平台化 + 效率革命 + 资本反身性解绑）分散成各自独立的 wildcard。本报告的首要价值，是把它们重新缝合。

第一，主导矛盾从"能否造出更大的训练 run"转为"推理负载的结构化吞噬"，训练与推理在 RL 闭环中边界融合——增量算力从堆参数 / token 转向 RL + 合成数据 + 推理时计算三条新轴。

第二，硬约束从硅（GPU / 制程）整体下移到电力与先进封装（CoWoS / HBM）——瓶颈从"能量够不够"（数据中心 2030 仅占全球电力约 3%）变为"特定地点 18–36 个月能否拿到 GW 级稳定功率"，即从 TWh 之争变为 GW / 年 交付速度之争。

第三，算力从工程对象升格为金融资产与主权战略储备：期货化 / 期权化抢购 + 20 国级算力储备 + 双轨（西方 TSMC / Nvidia / CUDA vs 中国 SMIC / 昇腾 / CANN）硬分化。

横贯所有维度的裂缝 · 置信度高

整个底座建立在一个从未被独立证明的承重假设上——"推理需求真实、持久、可货币化"。8 个维度的压测无一例外把它列为最脆弱点。2028–2030 的真正分水岭不是技术，而是这个假设在哪一年、以多陡的斜率被市场检验。

02算力负载图景 · 训练 vs 推理的结构性反转

方向判断（置信度 高）：推理在 2028 超过训练 + 非 AI 负载，成为数据中心最大电力来源；到 2030 占 AI 总算力多数。这是 RAND / Introl / Goldman 多家罕见一致的结论，2025→2028 段近乎确定。

agentic / test-time compute 乘数 · 本报告最需警示之处

原预测把单任务 token 放大写成连乘：reasoning 隐藏链 10–100x × agentic 多轮 5–10x × 并行采样 best-of-N 5–50x = 头部任务 20x→数千x。压测关键纠正（置信度高）：这是连乘关系，误差放大而非抵消，每个乘数都可能向均值回归一档。

  • test-time compute 收益曲线很可能是 S 型（对数收益）而非指数型已有 "overthinking" / inverse-scaling 证据：超过某 token 预算准确率反而下降。产品默认会从 "always-on reasoning" 退回 "router 按难度调用"，90% query 走短路径 → reasoning 放大均值从 10–100x 坍缩到实际 3–5x。
  • multi-agent 的 20–30x 常驻负载是商业化证据最薄弱的乘数Gartner 已预警约 40% agent 项目因成本超支被砍。若该乘数坍缩为 2–3x，2030 占比直接从 75% 掉到 55–60%。
不可通约单位陷阱 · 置信度高 · 务必纳入决策

token 量、FLOPs、电力是三种不可通约的单位。原预测用 token 计推理占比（75–80%），用 FLOPs / 瓦特计瓶颈（327GW），却把两者当互相印证。一旦 MoE 极致稀疏 + 推测解码 + 蒸馏把每 token FLOPs 压低 1–2 个数量级，"token 爆炸"与"电力爆炸"可同时为真、同时为假。
决策含义：不要用 token 增速给电力 / 资本支出做规划基线。

04三套场景

压测揭示乐观 / 悲观并非对称——下行剧本不需要任何技术失败或台海热战，只需 scaling 平台化 + 资本反身性解绑两个内生变量联动。因此基准与悲观的概率质量被原预测系统性低估。

场景 A · 乐观（超指数继续）概率 ≈ 20–25%
触发条件
test-time compute 收益维持类指数 + agentic ROI 2028 后转正规模化 + 327GW 真能建成 + 合成数据绕开数据墙且无 model collapse + 资本超周期不降速。需 4 个非独立乐观假设同时兑现。
算力总量
推理占 75–80%，配比 5:1+，前沿 run 接近 2e29，AI 数据中心 ~327GW，总负载 vs 2025 增长接近 2 个数量级。
架构特征
百万 GPU / 多 GW 跨大陆联合体常态化；1MW 机柜成默认；算力期货市场深度流动、调度台进 FinOps；双轨硬分化固化。
场景 B · 基准（约束下稳步） 本报告主推概率 ≈ 45–50%
触发条件
方向全对但量级被物理与经济摩擦削平——电网/变压器（18–36 月交付）成节流阀使部分机房"有楼无电"；TTC 部分饱和转自适应；agentic 回撤到"单 agent + 固定 DAG + 人审"；效率革命压低每 token FLOPs，token 涨但电力增速腰斩。
算力总量
推理占 60–70%，配比 2.5:1 — 3.5:1；前沿 run 3e28–1e29；AI 数据中心 200–250GW；全球 DC 耗电 ~945TWh（占全球 ~3%）大体成立但 AI 部分增速低于 Goldman 24x。
架构特征
液冷/800VDC/电力第一约束坐实；机柜密度商业最优停在 200–600kW，1MW 仅实验性；CPO 互连成主流但"光学计算"仍利基；容错 + 编译器可移植性兑现，算力金融化主线沦为巨头内部成本核算；双轨共存但中国受 HBM 良率（CXMT 5–20%）持续锁喉。
场景 C · 悲观（撞墙或泡沫破裂）概率 ≈ 25–30% · 被原预测低估
触发条件
不需技术失败 / 不需台海热战。2027H2 起：① TTC 撞对数收益墙 + ② agentic ROI 证伪（Gartner 砍单兑现为 60%+）+ ③ 某头部 hyperscaler 下调 AI capex 指引/计提数据中心减值，成为"ROI 证伪"谢林点 → 反身性循环融资（Nvidia→OpenAI→Azure→Nvidia）非线性反向解绑。叠加 DeepSeek 式效率阶跃使 2026 抢购产能在 2028 结构性过剩。
算力总量
Big-5 某年 capex 同比腰斩；全球 capex 从 $1.7T 预期下修到 $1T 以下；AI 数据中心电力落在 150–200GW（需求自我退烧，电网卡死预言因需求消失而从未触发）；推理仍超训练但占比停在 55–60%。
架构特征
封装 + HBM 从"供不应求 3x"翻转为双重产能堰塞湖 + 数十亿美元资本减记；H100 现货跌破 $10k，二手洪峰冲垮租赁价；HBM5 叙事因 HBM4e 已够用而无人接盘；算力期货因单边下行、套保枯竭而流动性崩塌；管制反转（放开 H200/B30A 对华）使中国自主链投资搁浅，双轨不是固化而是逆转。

07反共识判断（与主流叙事相左但有逻辑链）

  • 反共识 1 中-高 · "推理负载指数化吞噬一切"很可能是上界情景而非中位 真正的非线性风险在"算力总需求"与"单一训练实体规模/电力需求"的脱钩处，而非两者同步爆炸。即便总需求涨，它转向 inference 时对"单一逻辑训练实体"无要求——inference 要的是地理分散、靠近用户与廉价电力的中型节点舰队（50–200MW），而非跨大陆 multi-GW 联合体。含义：押注"百万卡巨集群成标准范式"风险高于押注"分布式中型 inference 舰队"。
  • 反共识 2 中 · 电力作为"最硬约束"可能从未被真正触发 不是因为产能赶上，而是因为需求被效率与经济双重压平，电网卡死的预言自我消解。若 MoE 稀疏 + FP4/FP2 + 蒸馏把单位智能 FLOPs 砍掉 40–60%，叠加 agentic ROI 退烧使需求自我退烧，2030 实际电力落在 150–200GW 而非 327GW——瓶颈消失不是因为供给胜利，而是因为需求退场。
  • 反共识 3 中 · "算力商品化金融市场"是 8 条主线中最可能流产的一条 GPU-hour 的异质性使其无法成为原油式可交割标准品。H100 vs B200 vs MI300 vs TPU vs Trainium 在 token/s、内存带宽、互连域、软件栈上差异巨大，同型号在不同拓扑/PUE/网络邻近性下有效产能也天差地别。现货指数可做，但"容量期权 + 跨厂可替代标准单位 + 调度器价格驱动放置"需要低基差可交割底层资产。更可能结局：算力交易台沦为少数巨头内部成本核算工具，而非真正的跨云套利市场。
报告置信度总评

方向性判断（推理 > 训练、瓶颈从 GPU 转向电力与封装、增量从堆 token 转向 RL + 合成数据、双轨硬分化、液冷/800VDC/CPO 成主流）= 高

2030 具体量级（配比、token/日、GW、占比、capex 总额）= 中，应作区间与场景化使用而非单点外推

算力金融化主线 + 百万卡巨集群范式 = 低-中，建议当上界情景对待

最大系统性风险：8 维度共享的"推理需求可货币化"承重假设从未被独立证明，其失效路径（scaling 平台化 + 效率革命 + 资本反身性解绑）被各维度人为拆散成独立 wildcard，联合发生概率被结构性低估——本报告已将其重新缝合为场景 C 并上调至约 25–30% 概率质量。

主导矛盾不再是"能不能造出更大的训练 run"，而是"推理负载的指数化吞噬一切"。前沿训练 run 仍按 ~4–5x/年 推进至接近 1e29 FLOP，但撞数据墙后增量算力从堆参数/token 转向 RL + 合成数据 + 推理时计算三条新轴。真正爆炸的是推理：agentic 长链路 + 多轮 + 并行采样使单任务 token 较 2023 放大 10–100x，推理:训练 从 1:1（2025）演变为 5:1（2030），推理占总算力 75–80%，瓶颈从 GPU 转为电力。
逐年演进
2028
前沿 run 1e27–5e27 FLOP；>1e26 模型约 80 个。数据墙 front edge 到达，合成数据 + RL 成增量主力。推理:训练 ≈ 3:1，占总算力 ~65–70%。全球推理 ~5–15 quadrillion token/日。AI 数据中心 ~100–130GW。
2029
前沿 run 5e27–3e28；增量训练算力 50%+ 投入 RL rollout（本质大规模推理）。TTC 成差异化主战场，头部任务推到 10万–百万 token。推理:训练 ≈ 4:1，占 ~72%。AI 数据中心 ~180–240GW，电力成首要约束。
2030
前沿 run 接近 1e29（乐观 2e29）。训练与推理在 RL 闭环深度融合。推理:训练 ≈ 5:1 或更高，占 75–80%。全球推理 ~100+ quadrillion token/日（1e17 级）。AI 数据中心 ~300–330GW。总负载 vs 2025 增长 1–2 个数量级。
关键量化
  • 前沿单 run FLOPs：2025 ~5e26（Grok 4）→ 2030 接近 1e29，依据 Epoch 历史 4.7x/年 + 5GW 园区供电测算
  • 全球推理 token/日：2026 全行业 ~0.5–1 quadrillion → 2030 ~100+ quadrillion（Google 7x/年 + Goldman 24x@2026→2030）
  • 单任务 token 放大：reasoning 链 10–100x × agentic 5–10x × best-of-N 5–50x ⇒ 头部 20x→数千x
  • AI 数据中心电力：2024 ~11GW → 2027 ~68GW → 2030 ~327GW（RAND）；Stargate 10GW、xAI Colossus 2 ~1GW/100万GPU
黑天鹅
  • latent/recurrent-depth reasoning 使 TTC token 成本骤降 1–2 数量级，推理曲线被效率革命压平
  • 电网成硬天花板：327GW 物理上建不出，token 增速远低于 24x，靠蒸馏+边缘+缓存满足
  • 成本反噬：token 计费让 agentic ROI 转负（Microsoft 撤 Claude Code 是早期信号），2028–29 出现"AI 泡沫"式回调
  • TTC 撞收益递减墙：超过某 token 预算准确率不再提升，best-of-N 性价比崩溃，头部任务 token 2029 见顶
压测 plausible反向剧本：2028 Agentic ROI 证伪 + 效率革命双杀
2027H2 起企业发现长链路 best-of-N 边际准确率在中等 token 预算后迅速饱和，token 计费让单任务成本逼近人工。2028H1 出现第一次"agentic 泡沫"回调，转向自适应 TTC（默认廉价快答）。同时 MoE 稀疏 + 推测解码 + latent reasoning 使每单位智能 FLOPs 降 1–1.5 数量级，60–70% 推理量迁到边缘/缓存。结果：token 计数仍涨（也许真到 24x），但 FLOPs 负载与电力严重背离——2030 实际电力落在 150–200GW 而非 327GW，电网卡死预言因需求自我退烧而从未触发。推理:训练 稳定在 2.5–3:1 而非 5:1。杀伤力在于：不需 scaling law 戏剧性失效，只需 TTC 是 S 型 + 经济护栏早咬合 1–2 年。
最脆弱（复合）：TTC scaling 不失效 ∧ agentic ROI 转正——二者是同一赌注的两面，预测自己在 drivers/wildcards 里已埋反证却选择性忽略
量级混淆：核心 KPI 用 token 计，瓶颈用 FLOPs/瓦特计，两条曲线用不可通约单位却被当互证
单点信源：RAND/Goldman/Epoch 高度相关、彼此循环引用，诞生于资本狂热期，存在系统性同向偏差
调整后置信：中（偏低）。方向性判断稳健（2025→2028 段大概率成立）；但 5:1 配比、100+ quadrillion、327GW 三个拉向"指数化"的量化同时依赖 4 个非独立乐观假设全部兑现。建议 thesis 改写为范围预测：推理占比 60–80%、配比 2.5:1–5:1。
D2 · SILICON加速器硬件底座：制程 / 先进封装 / HBM / 竞争格局 / 互连中高▶
旗舰算力增长引擎从"晶体管制程"彻底切换到"先进封装 + HBM + 互连"三位一体。A14/A16 仅贡献 ~15–25%/节点的密度红利，真正决定单封装算力上限的是 CoWoS/CoPoS reticle 拼接倍数（5.5x→14x+）与 HBM4e/HBM5 容量带宽墙。瓶颈从光刻转移到封装产能（CoWoS 供不应求 ~3x）与 HBM 良率。NVIDIA 守住训练制高点但推理份额从 90%+ 被 ASIC 侵蚀至 20–30%；中国昇腾走 SMIC 7nm DUV + 自研 HBM "以量补质"独立路线，被封装与 HBM 而非制程卡住。
逐年演进
2028
A14 量产、A16 成 HPC 主力。CoWoS 14x reticle 量产，CoPoS 面板级爬坡，glass substrate 试产。HBM4e 全面铺开（16-Hi，单 stack ~3.6TB/s），旗舰封装 HBM 1–1.5TB。NVIDIA 推理份额跌至 20–30%。单封装 FP4 ~150–250 PFLOPS，~4–4.5kW。
2029
A13/A12 登场但无 high-NA EUV。CoWoS >14x、CoPoS 量产，glass substrate 规模采用。HBM5 早期样品，旗舰 HBM 1.5–2.5TB@40–60TB/s。训练-推理硬件分化加深，wafer-scale 在利基扩大。CPO 成旗舰标配选项。单封装 FP4 ~250–400 PFLOPS，~4.5–5.5kW。
2030
面板级 + glass substrate 成旗舰标准，3D 逻辑堆叠普及，单封装等效面积远超单 reticle。HBM5 量产，旗舰封装内存 2–4TB@60–100TB/s。三足定型。CPO 全面进入封装，光成 scale-up/out 默认介质。单封装 FP4 ~400–800 PFLOPS，~5–6kW（wafer-scale 达 10kW+）。
关键量化
  • 旗舰 FP4：Rubin 50 PFLOPS（2026）→ 2030 ~400–800 PFLOPS
  • HBM：Rubin 288GB@13TB/s → Rubin Ultra 1TB@~32TB/s → 2030 旗舰 2–4TB@60–100TB/s
  • CoWoS 产能：~95K WPM（2026/01）仍供不应求约 3x，2026 计划 +33% 仍为第一瓶颈
  • 竞争：NVIDIA 推理份额 90%+→20–30%（2028）；定制 ASIC ~44.6% CAGR；超大规模 capex $660–690B（2026）
  • 中国：昇腾 910C SMIC 7nm DUV，2026 目标 ~60 万颗；950PR→960（2027Q4）→970（2028Q4）
黑天鹅
  • HBM 良率事故或 16-Hi 混合键合翘曲，把所有旗舰算力上量集体推迟 1–2 年
  • CPO 可靠性（激光寿命/现场可换）不达标，光进封装推迟，scale-up 带宽墙提前到来
  • 推理范式转移（线性注意力/FP4→FP2/三值）使内存带宽需求结构性下降，重定义硬件优先级
  • 电力成比芯片更硬的约束——单机柜逼近 1MW，供电而非芯片产能决定实际部署上限
压测 plausible反向剧本：2027H2 需求证伪 → 双重产能堰塞湖
2027H2 押注的 agentic/推理收入未兑现单位经济性，一两家超大规模厂商因 AI 折旧下修 capex，触发同向砍单。封装/HBM 是 12–18 月 lead-time 重资产，需求转向 2028 才传导到产线——此时 CoWoS 扩产与 HBM 16-Hi 新线刚投产，撞上需求下行，从"供不应求 3x"瞬间翻转为两位数产能空转。叠加极致量化把单位智能内存带宽需求砍 40–60%，客户发现 HBM4e 已够用，HBM5 溢价无人接盘。第三重打击：1MW 机柜因变电站审批滞后无法满载。结果 2028–29 是封装+HBM 双重过剩 + 数十亿美元资本减记 + 节点主动放缓。不需任何技术失败，只需"需求线性增长"承重假设松动。
最脆弱（根基）：AI capex 超级周期延续无需求断崖——把"供给瓶颈"当主角，几乎没为"需求瓶颈"留对称剧本
隐藏同向相关：电力/液冷"跟得上"与 capex 假设实为负相关却被当独立；电网 3–7 年 lead-time 塞进 2 年乐观跟随项
单点垄断脆性：TSMC + Broadcom/Marvell 95% 集中，尾部风险是相关、可同时触发的
调整后置信：中。技术 roadmap 层面偏高（已公布工程路线，"若需求成立则技术能交付"稳健）；但对需求侧/电力侧做了系统性单边乐观，更像"供给侧能力上界 roadmap"而非"需求加权实际落地预测"。
D3 · DATACENTER数据中心物理底座：集群 / 液冷 / 机柜密度 / 网络 / 跨域训练 / 电力中▶
单训练实体从"10万卡/百MW 单集群"跃迁到"百万 GPU / 多 GW 跨园区联合体"。物理形态被三件事重定义：直接芯片液冷（冷板→两相/局部浸没）成默认、机柜密度从 ~130kW 走向 600kW（Kyber/2027）并在 Feynman 代逼近 1MW、800VDC 高压直流取代传统交流配电。单 AC 大楼无法供电的体量迫使"多数据中心异步分布式训练"成前沿标准范式，电力（而非土地或硅）成选址第一硬约束——GW 级电力交付前置周期（3–5 年）决定谁能训练下一代模型。
逐年演进
2028
Rubin Ultra（Kyber/NVL576, 600kW 机柜）规模部署，800VDC + 全液冷（零风扇）成新建默认。单园区 50万–100万卡/1–2GW。前沿训练首次常态化跨 2–3 园区异步。scale-up 域 144 封装/576 die。PUE 主流 1.1–1.15。
2029
Feynman 代登场，机柜向 800kW–1MW；两相直液 + 混合浸没进入生产。百万 GPU 等效集群（跨 3–5 园区、合计 3–5GW）。CPO/硅光进 scale-up 域缓解铜距离极限，OCS 在 scale-out 规模化。自建发电（燃气快上、SMR 签约、光储）成选址标配。
2030
GW→多 GW 单一逻辑训练实体常态化，"AI 工厂"按 GW 采购。1MW 级机柜在最先进设施落地。per-GPU scale-up 带宽 7–10+ TB/s。跨大陆分层异步训练成熟。电力成绝对第一约束，部分算力"就电而建"拉到偏远地区。PUE 头部逼近 1.08–1.10。
关键量化
  • 单训练集群：2028 单园区 50万–100万卡/1–2GW；2030 跨园区 100万–数百万 GPU 等效/3–10GW
  • 机柜密度：当前 ~120–140kW → Kyber 600kW（2027/28）→ Feynman ~800kW–1MW（2029/30）
  • per-GPU NVLink：1.8TB/s（Blackwell）→3.6TB/s（Rubin）→7+TB/s（2029/30）；CPO 较可插拔光模块降功耗 ~3.5x
  • 配电：交流→800VDC（端到端效率 +~5%，运维成本 -~70%）；45°C 高温液冷
  • 电力前置期：GW 级变电站/输电 3–5 年交付，2030 前部分园区靠自建燃气/SMR/光储补缺
黑天鹅
  • 算法效率突破刺破"百万卡集群"军备竞赛，密度/功率曲线提前平台化
  • 电力交付硬卡死：已建机房"有楼无电"，GW 级集群上线推迟 2–3 年
  • 跨地域异步训练收敛性不及预期，被迫退回单园区同步，反而强化单点 GW 供电压力
  • 两相液冷/1MW 机柜可靠性或供应链出问题，密度爬坡卡在 ~600kW 而非 1MW
压测 plausible承重墙：scaling 关系延续 2030 + 异步训练成熟
最脆弱是"每代算力需求显著增长且不被算法效率抵消"——这是两个独立趋势的差值假设。只要算法效率（MoE 稀疏、FP4→FP2、蒸馏、inference-time scaling 替代 pretraining）斜率接近硬件 FLOPs 增长斜率，"更大集群=更强模型"即被打破。2024–26 已现 pretraining 边际收益放缓信号。关键混淆：把"算力总需求"与"单一训练实体规模"混为一谈——即使总需求涨，若转移到 inference 而非单体 training，"百万卡单一逻辑训练实体"根本不需要存在。次脆弱：跨地域异步训练面临梯度陈旧/收敛退化/容错爆炸，是技术不确定性最高却被当既定事实的一环；若失败，"多 GW 跨大陆联合体"核心形态直接坍塌。
结构性脆弱：把单一厂商（NVIDIA）roadmap 当物理定律——节奏滑期 + 生态分叉（TPU 用 OCS+自有配电）+ 1MW 机柜可能长期卡在 400–600kW 经济最优点
调整后置信：中。液冷/800VDC/电力第一约束的方向稳健；但机柜密度商业最优、百万卡单体范式、异步训练成熟度三处高度依赖未证假设，建议作场景化使用。
D4 · INFERENCE推理范式重塑算力形态：TTC / agent 常驻 / KV-cache / 分离式 / 端侧分流中▶
test-time compute / 长 CoT / 树搜索 / 并行采样把推理变成"小训练"；multi-agent 常驻带来持续负载基线；KV-cache 成内存与带宽核心瓶颈，催生分离式架构（prefill/decode disaggregation）、KV 分层与卸载；推测解码 + MoE 稀疏 + FP4/INT4 量化普及；端侧 NPU 承接部分推理。到 2030 推理占 AI 算力 70–75%（压测下修：上沿 75、中位 55–65），"带宽中心/KV 中心"成转型方向。
压测核心结论
  • 最脆弱（循环依赖）：TTC 持续有效且经济——这是整个 thesis 的承重墙（70–75% 占比、token 10–100x×20–30x 放大全由它支撑）。但长 CoT 是对数收益（已有 overthinking/inverse-scaling 证据），成本是线性的。一旦边际收益跌破企业可感知阈值，TTC 从"默认开启"退回"仅难任务开启"，放大倍数从 10–100x 坍缩到 2–5x。
  • 商业未证：multi-agent 24/7 常驻负载贡献 20–30x 放大，但 Gartner ~40% agent 项目因成本超支被砍。若坍缩为 2–3x，2030 占比从 70–75% 掉到 ~55–60%。
  • 非对称风险：Transformer + KV-cache 若被 SSM/线性注意力在 2028–29 大规模商用替代，KV 池化、P/D 分离、CXL 分层这些"标配 playbook"重资本投资集体减值。
压测 plausible反向剧本：scaling 失效 + 需求祛魅的"温水坍缩"（2028–29）
不是戏剧性崩盘，而是三条曲线同时让步。(1) TTC 撞对数收益墙：2027–28 头部发现 o 系列后续靠堆 token 的增益跌到个位数%，产品默认改为 router 按难度调用，90% query 走短路径，token 放大人口加权均值从 10–100x 坍缩到 3–5x。(2) multi-agent 证伪：Gartner 40% 砍单兑现为 60%+，回退到"单 agent + 固定工作流 + 人审"，24/7 基线从未规模化，20–30x 乘数蒸发。(3) 需求祛魅：资本市场对"token 用量无限增长"重新定价，2028 capex 从 +60%/年降到 +15%/年。结果：推理仍超训练（方向对），但占 AI 总算力停在 55–60% 而非 70–75%，"带宽中心/KV 中心"转型放缓。叠加某线性注意力架构 2029 商用，从根上削弱 KV-cache 瓶颈，使前两年 KV 重资本投资部分减值。不需任何单一黑天鹅，只要三个最关键乘数各向均值回归一档。
调整后置信：中。方向性论断稳健（2028 推理>训练成最大电力源、$/Mtok 年降速放缓、FP4 普及、端侧分流真实、P/D 分离头部落地）；但承重的量化（70–75% 占比、token 连乘、Jevons 必推升总支出）建立在三个相互强化的乘数假设上，连乘误差放大而非抵消。建议把 70–75% 当上沿、55–65% 当中位。
D5 · ENERGY能源 / 电力底座：电力供应 / 电网接入 / 发电组合（最硬约束）中▶
电力（而非芯片）是最硬约束。瓶颈不在"全球有没有电"（数据中心 2030 仅占全球电力约 3%，绝对量可承受），而在"特定地点能否在 18–36 个月内拿到 GW 级稳定功率"——电网接入排队（2000+ GW 待并网、time-to-power 拉长到 5–7 年）把矛盾从能量（TWh）转移到产能交付速度（GW/年）与选址。行业绕过电网：自备电（BTM 天然气）做 2028 前过渡桥梁，核电复兴（重启退役机组 + SMR）做 2029 后清洁基荷，算力随电迁移到得州/中东/北欧，PPA 与发电机组档期被金融化抢购。
逐年演进
2028
天然气自备电主导过渡期。BTM 燃气电厂（18 个月可投产）成 GW 级新增算力主力电源。电网接入仍是最大瓶颈，热点排队 5–7 年。首批重启核电（Three Mile Island/Crane ~835MW、Palisades）满负荷供 MS/Meta。全球 DC 耗电 ~600–700 TWh。
2029

D6 · SYSTEMSTACK调度与系统软件底座：容错训练 / 编译器 / 通信库 / 分布式并行 / 算力商品化中▶
系统软件底座从"人工调优、单厂栈"转向三条主线并进：① 故障常态化下的自愈编排——百万卡故障频率使容错从可选优化升级为第一性约束（异步/分层 checkpoint + 弹性重构 + 部分恢复成默认），单点故障不再中断训练；② 跨硬件可移植抽象——CUDA 护城河在编译器层（Triton/MLIR）被持续侵蚀但不被攻破，护城河上移到"超优化 kernel 库 + 通信库 + NVLink 域硬件协同"；③ 算力商品化金融市场——CME/ICE 2026 启动期货化，到 2028–30 形成现货指数 + 期货 + 容量期权完整风险转移层，调度器消费市场价格做跨地域套利。

D7 · NEWPARADIGM非主流计算范式：硅光 / 存内 / 模拟 / 神经形态 / 超导 / 量子 / 生物中▶
2028–2030 的"非主流"增量几乎全部来自三条已上量产路径：① 硅光互连 + co-packaged optics 先成主流（打破 I/O 墙），光学矩阵乘加速器在推理边缘小规模商用；② 存内/近存计算以 RRAM/SRAM CIM 形态进入边缘 AI 与端侧 SoC；③ 神经形态（Loihi/NorthPole）在事件驱动的机器人/传感/Physical-AI 利基商用。超导属"晚期实验室→早期原型"，2030 前不进主流。量子对 AI 训练/推理算力实质无贡献。DNA/生物计算 2030 前只在归档存储。核心信号：它们都不会"替代"GPU，而是以异构协处理器/互连/利基加速器身份嵌入。
