## 三、迭代计划

### 迭代 1: 模板库扩充（P0）

**目标**: 从 10 个模板扩充到 30+ 个，并增加完整元数据

**具体任务**:

| #   | 任务                        | 产出          | 工作量 |
| --- | --------------------------- | ------------- | ------ |
| 1.1 | 学习 Genspark HTML 输出模式 | 设计规范文档  | 0.5天  |
| 1.2 | 创建叙事型模板 (5个)        | N-001 ~ N-005 | 1天    |
| 1.3 | 创建结构型模板 (9个)        | S-001 ~ S-009 | 1.5天  |
| 1.4 | 创建数据型模板 (6个)        | D-001 ~ D-006 | 1天    |
| 1.5 | 创建内容型模板 (7个)        | C-001 ~ C-007 | 1天    |
| 1.6 | 创建行动型模板 (5个)        | A-001 ~ A-005 | 0.5天  |
| 1.7 | 为每个模板添加元数据        | 完整 metadata | 1天    |

**模板元数据结构**:

```typescript
interface TemplateMetadata {
  id: string; // "S-003"
  name: string; // "三支柱"
  category: string; // "structural"
  keywords: string[]; // 语义匹配关键词
  useCases: string[]; // 适用场景描述
  contentSpec: {
    minBlocks: number;
    maxBlocks: number;
  };
  positionFit: {
    // 叙事位置适配
    opening: number; // 0-1
    middle: number;
    closing: number;
  };
}
```

**一致性检查项**:

```typescript
interface ConsistencyCheck {
  // 风格
  colorPalette: boolean; // 颜色是否统一
  fontFamily: boolean; // 字体是否统一
  cardStyle: boolean; // 卡片风格是否统一

  // 语言
  terminology: boolean; // 术语是否统一
  personPerspective: boolean; // 人称是否统一（我们 vs 公司）
  tone: boolean; // 语气是否统一

  // 结构
  headerFormat: boolean; // 标题格式是否统一
  footerFormat: boolean; // 页脚是否统一
  pageNumbering: boolean; // 页码是否连续
}
```

### 迭代 4: 提示词优化（P1）

**目标**: 优化 5 个核心 Skill 的提示词

**具体任务**:

| #   | 任务                | 当前问题       | 改进方向           |
| --- | ------------------- | -------------- | ------------------ |
| 4.1 | task-decomposition  | 意图理解不够深 | 增加追问、Few-shot |
| 4.2 | outline-planning    | 叙事结构单一   | 增加多种叙事模式   |
| 4.3 | page-type-selection | 依赖规则       | 增加 AI 辅助判断   |
| 4.4 | four-step-design    | 设计思考不透明 | 增加思考过程输出   |
| 4.5 | content-compression | 内容过于压缩   | 调整压缩策略       |

**提示词优化原则**:

1. **Few-shot**: 每个 Skill 2-3 个高质量示例
2. **Chain-of-Thought**: 先分析再输出
3. **结构化输出**: JSON 格式
4. **自我评估**: 输出前评分 (1-10)
5. **负面示例**: 明确"不要做什么"

## 六、风险与缓解

| 风险           | 影响         | 缓解措施                     |
| -------------- | ------------ | ---------------------------- |
| 模板质量不高   | 生成效果差   | 参考 Genspark 输出，逐个打磨 |
| 匹配算法不准   | 模板选择错误 | 保留规则兜底，渐进优化       |
| 一致性检查误报 | 错误修改     | 先警告不自动修复，积累数据   |
| 提示词改坏     | 生成质量下降 | A/B 测试，保留旧版本         |
