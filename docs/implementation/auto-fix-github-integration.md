# GitHub + Claude Code 自动修复集成方案

## 概述

当 Triage Agent 判定反馈可以自动修复 (`action: "auto_fix"`) 时，自动触发 Claude Code 进行代码修复并创建 PR。

## 架构设计

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Feedback  │────▶│   Triage    │────▶│   GitHub    │────▶│ GitHub      │
│   Service   │     │   Agent     │     │   Issue     │     │ Actions     │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
                           │                                       │
                           │ auto_fix                              │
                           ▼                                       ▼
                    ┌─────────────┐                         ┌─────────────┐
                    │  Create     │                         │ Claude Code │
                    │  Issue      │                         │    CLI      │
                    └─────────────┘                         └─────────────┘
                                                                   │
                                                                   ▼
                                                            ┌─────────────┐
                                                            │  Create PR  │
                                                            └─────────────┘
```

## 实现方案

### Phase 1: 创建 GitHub Issue

当 `auto_fix` 决策产生时，自动创建 GitHub Issue：

```typescript
// backend/src/modules/core/feedback/github/github-issue.service.ts
@Injectable()
export class GitHubIssueService {
  async createAutoFixIssue(
    feedbackId: string,
    decision: TriageDecision,
  ): Promise<string> {
    const issueBody = this.buildIssueBody(feedbackId, decision);

    // 使用 GitHub API 创建 Issue
    const response = await this.octokit.issues.create({
      owner: "your-org",
      repo: "deepdive",
      title: `[Auto-Fix] ${decision.classification.affectedModule}: ${feedbackId}`,
      body: issueBody,
      labels: ["auto-fix-candidate", "ai-generated", decision.priority.level],
    });

    return response.data.html_url;
  }
}
```

### Phase 2: GitHub Actions 触发 Claude Code

```yaml
# .github/workflows/auto-fix.yml
name: Auto Fix with Claude Code

on:
  issues:
    types: [opened, labeled]

jobs:
  auto-fix:
    if: contains(github.event.issue.labels.*.name, 'auto-fix-candidate')
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: "20"

      - name: Install Claude Code
        run: npm install -g @anthropic-ai/claude-code

      - name: Extract Fix Instructions
        id: extract
        run: |
          # 从 Issue body 中提取修复指令
          INSTRUCTIONS=$(echo "${{ github.event.issue.body }}" | grep -A 100 "## Fix Instructions" | head -50)
          echo "instructions<<EOF" >> $GITHUB_OUTPUT
          echo "$INSTRUCTIONS" >> $GITHUB_OUTPUT
          echo "EOF" >> $GITHUB_OUTPUT

      - name: Run Claude Code
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        run: |
          # 创建修复分支
          BRANCH_NAME="auto-fix/issue-${{ github.event.issue.number }}"
          git checkout -b $BRANCH_NAME

          # 运行 Claude Code (非交互模式)
          claude-code --print --dangerously-skip-permissions \
            --prompt "${{ steps.extract.outputs.instructions }}"

          # 提交更改
          git config user.name "Claude Code Bot"
          git config user.email "claude-code-bot@example.com"
          git add .
          git commit -m "fix: auto-fix for issue #${{ github.event.issue.number }}"
          git push origin $BRANCH_NAME

      - name: Create Pull Request
        uses: peter-evans/create-pull-request@v5
        with:
          token: ${{ secrets.GITHUB_TOKEN }}
          branch: auto-fix/issue-${{ github.event.issue.number }}
          title: "[Auto-Fix] Fixes #${{ github.event.issue.number }}"
          body: |
            ## Auto-generated Fix

            This PR was automatically generated by Claude Code to fix issue #${{ github.event.issue.number }}.

            ### Changes
            - See commits for detailed changes

            ### Review Notes
            - Please review carefully before merging
            - Run tests to verify the fix

            ---
            🤖 Generated by Claude Code
          labels: |
            auto-fix
            needs-review
```

### Phase 3: 后端集成

```typescript
// backend/src/modules/core/feedback/auto-fix/auto-fix.service.ts
@Injectable()
export class AutoFixService {
  constructor(
    private readonly githubService: GitHubIssueService,
    private readonly eventEmitter: EventEmitter2,
  ) {}

  /**
   * 触发自动修复流程
   */
  async triggerAutoFix(
    feedbackId: string,
    decision: TriageDecision,
  ): Promise<{ issueUrl: string; status: string }> {
    // 1. 创建 GitHub Issue
    const issueUrl = await this.githubService.createAutoFixIssue(
      feedbackId,
      decision,
    );

    // 2. 更新反馈状态
    this.eventEmitter.emit(FeedbackEvent.FIX_STARTED, {
      feedbackId,
      fixType: "auto",
      issueUrl,
      startedAt: new Date(),
    });

    // 3. GitHub Actions 会自动触发
    return {
      issueUrl,
      status: "pending", // GitHub Actions 正在处理
    };
  }
}
```

## 安全考虑

### 1. 权限控制

- Claude Code 在隔离环境中运行
- 只能访问指定的代码目录
- 无法访问生产数据库或密钥

### 2. 代码审查

- 所有自动生成的 PR 需要人工审查
- 设置 `needs-review` 标签
- CI/CD 检查必须通过

### 3. 限制范围

- 只允许修改特定目录
- 禁止修改敏感文件 (.env, secrets, etc.)
- 设置文件修改数量上限

### 回滚机制

```typescript
async rollbackAutoFix(prNumber: number): Promise<void> {
  // 1. 关闭 PR
  await this.octokit.pulls.update({
    owner: 'your-org',
    repo: 'deepdive',
    pull_number: prNumber,
    state: 'closed',
  });

  // 2. 删除分支
  await this.octokit.git.deleteRef({
    owner: 'your-org',
    repo: 'deepdive',
    ref: `heads/auto-fix/issue-${prNumber}`,
  });

  // 3. 更新反馈状态
  this.eventEmitter.emit(FeedbackEvent.FIX_FAILED, {
    feedbackId,
    reason: 'Rolled back by admin',
  });
}
```

## 替代方案

### 方案 B: 自托管 Claude Code 服务

如果不想使用 GitHub Actions，可以自托管 Claude Code：

```typescript
// 使用 Child Process 调用 Claude Code CLI
async runClaudeCode(prompt: string): Promise<string> {
  const result = await exec(`claude-code --print --prompt "${prompt}"`, {
    env: { ANTHROPIC_API_KEY: process.env.ANTHROPIC_API_KEY },
    cwd: '/path/to/repo',
  });
  return result.stdout;
}
```

### 方案 C: 使用 Claude Agent SDK

```typescript
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic();

// 使用 tool_use 让 Claude 直接编辑文件
const response = await client.messages.create({
  model: "claude-sonnet-4-20250514",
  max_tokens: 4096,
  tools: [
    {
      name: "edit_file",
      description: "Edit a file in the codebase",
      input_schema: {
        type: "object",
        properties: {
          file_path: { type: "string" },
          changes: { type: "string" },
        },
      },
    },
  ],
  messages: [{ role: "user", content: prompt }],
});
```
