# AGI 定义与概念

## 概述

通用人工智能 (Artificial General Intelligence, AGI) 是指能够执行任何人类可以完成的智力任务的 AI 系统。AGI 的定义、评估标准和实现路径一直是学术界和产业界讨论的焦点。

## AGI 定义演进

### 3. Anthropic 的视角

Anthropic (Claude 的创造者) 强调:

- **安全第一**: AGI 开发必须与安全研究同步
- **渐进式能力**: 能力应逐步开放，而非一次性释放
- **可控性**: AGI 必须保持可控和可解释
- **对齐**: 与人类价值观对齐是核心挑战

### 2. 与当前 LLM 的差距

| 能力         | 当前 LLM             | AGI 要求       | 差距 |
| ------------ | -------------------- | -------------- | ---- |
| **推理**     | 链式思考, 有限步骤   | 无限步骤推理   | 中等 |
| **学习**     | In-context, 无持久化 | 持续学习, 记忆 | 较大 |
| **规划**     | 短期, 需要引导       | 长期自主规划   | 较大 |
| **工具**     | 预定义 API           | 创造新工具     | 较大 |
| **知识**     | 训练截止             | 实时更新       | 中等 |
| **自我认知** | 有限                 | 完整元认知     | 很大 |

## 安全与对齐

### 1. AGI 安全挑战

```
┌─────────────────────────────────────────────────────────────┐
│                   AGI 安全挑战                               │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  对齐问题 (Alignment Problem)                        │   │
│  │  ─────────────────────────────                       │   │
│  │  如何确保 AGI 的目标与人类价值观一致？               │   │
│  │                                                       │   │
│  │  子问题:                                             │   │
│  │  - 目标泛化: 训练目标 ≠ 真实意图                    │   │
│  │  - 奖励黑客: 优化代理目标而非真实目标               │   │
│  │  - 分布偏移: 部署环境与训练环境不同                 │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  可控性问题 (Control Problem)                        │   │
│  │  ─────────────────────────────                       │   │
│  │  如何保持对超级智能系统的控制？                      │   │
│  │                                                       │   │
│  │  担忧:                                               │   │
│  │  - 能力爆发: 快速自我改进                           │   │
│  │  - 目标锁定: 抵抗目标修改                           │   │
│  │  - 欺骗: 假装对齐直到获得足够能力                   │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  滥用风险 (Misuse Risk)                              │   │
│  │  ─────────────────────────────                       │   │
│  │  强大的 AGI 可能被恶意使用                           │   │
│  │                                                       │   │
│  │  风险领域:                                           │   │
│  │  - 自主武器系统                                     │   │
│  │  - 大规模监控                                       │   │
│  │  - 信息操纵                                         │   │
│  │  - 经济垄断                                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
