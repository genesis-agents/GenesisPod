#### 需要重命名的文件

1. `docs/archive/2025-q1/reports/DELIVERY-MANIFEST.txt`
   - 问题: 全大写文件名
   - 建议: 重命名为 `delivery-manifest.txt`

2. `docs/prd/ai_studio_optimization.md`
   - 问题: 使用下划线
   - 建议: 重命名为 `ai-studio-optimization.md`

### 4.3 建议措施

创建自动化验证脚本：

```bash
# scripts/docs-validation.sh
#!/bin/bash

echo "检查文档命名规范..."

# 查找不符合规范的Markdown文件
find docs -name "*.md" -type f | while read file; do
  filename=$(basename "$file")

  # 检查是否包含大写字母
  if echo "$filename" | grep -q '[A-Z]'; then
    echo "❌ 违规: $file (包含大写字母)"
  fi

  # 检查是否包含下划线
  if echo "$filename" | grep -q '_'; then
    echo "❌ 违规: $file (包含下划线)"
  fi

  # 检查是否包含空格
  if echo "$filename" | grep -q ' '; then
    echo "❌ 违规: $file (包含空格)"
  fi
done

echo "✅ 检查完成"
```
