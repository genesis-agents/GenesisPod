## 1. 概述

### 1.1 系统定位

知识库系统是 GenesisPod 的核心基础能力模块，它不仅是一个独立的功能模块，更是为整个平台提供知识检索和上下文增强能力的基础设施。

**核心价值主张**：

```
"将分散的数据源统一为可检索的知识资产，让 AI 拥有用户的专属知识"
```

### 1.3 核心目标

| Goal                 | Description                  | Success Metric     |
| -------------------- | ---------------------------- | ------------------ |
| **统一知识管理**     | 整合多数据源为统一的知识库   | 支持 5+ 数据源类型 |
| **基础能力复用**     | 知识库能力被其他 AI 模块调用 | 4+ 模块集成        |
| **灵活检索**         | 支持混合检索和精排           | 检索相关度 > 90%   |
| **配置化 Embedding** | Embedding 模型可在 UI 配置   | 无硬编码模型       |

### 3.4 知识库引用机制

知识库作为基础能力，需要被其他模块复用。核心组件是 **知识库选择器 (KnowledgeBaseSelector)**。

#### 3.4.1 知识库选择器组件

```typescript
// 知识库选择器接口定义
interface KnowledgeBaseSelectorProps {
  // 选择模式
  mode: "single" | "multiple";

  // 已选中的知识库 ID 列表
  selectedIds: string[];

  // 选择变更回调
  onChange: (ids: string[]) => void;

  // 过滤条件
  filter?: {
    type?: "personal" | "team" | "all";
    status?: "READY" | "all";
  };

  // UI 变体
  variant?: "dropdown" | "modal" | "inline";

  // 是否显示搜索
  showSearch?: boolean;

  // 占位文本
  placeholder?: string;
}
```

#### 7.1.4 代码实现

```typescript
// backend/src/modules/ai/rag/services/embedding.service.ts

@Injectable()
export class EmbeddingService {
  private readonly logger = new Logger(EmbeddingService.name);

  constructor(
    private readonly prisma: PrismaService,
    private readonly adminService: AdminService,
  ) {}

  /**
   * 获取当前配置的 Embedding 模型
   */
  async getEmbeddingModel(): Promise<AIModel> {
    // 从 AIModel 表获取默认的 EMBEDDING 类型模型
    const model = await this.prisma.aIModel.findFirst({
      where: {
        isEnabled: true,
        isDefault: true,
        modelType: "EMBEDDING",
      },
    });

    if (!model) {
      // 回退到硬编码默认值
      this.logger.warn("No default embedding model configured, using fallback");
      return {
        modelId: "text-embedding-3-small",
        provider: "OpenAI",
        embeddingDimensions: 1536,
        maxInputTokens: 8191,
      } as any;
    }

    return model;
  }

  /**
   * 生成 Embedding
   */
  async generateEmbedding(text: string): Promise<EmbeddingResult> {
    const model = await this.getEmbeddingModel();
    const apiKey = await this.adminService.getOpenAIApiKey();

    const openai = new OpenAI({ apiKey });

    const response = await openai.embeddings.create({
      model: model.modelId,
      input: text,
    });

    return {
      embedding: response.data[0].embedding,
      model: model.modelId,
      dimensions: model.embeddingDimensions,
      tokenCount: response.usage?.total_tokens || 0,
    };
  }
}
```

#### 7.2.4 统一 RAG 查询 API

```yaml
# 统一 RAG 查询接口 (供所有模块调用)
POST   /api/v1/rag/query
```

**请求体**：

```typescript
interface RAGQueryRequest {
  query: string; // 查询文本
  knowledgeBaseIds: string[]; // 知识库 ID 列表
  options?: {
    topK?: number; // 返回结果数量 (默认 5)
    enableHyDE?: boolean; // 启用 HyDE (默认 false)
    enableRerank?: boolean; // 启用 Rerank (默认 true)
    bm25Weight?: number; // BM25 权重 (默认 0.5)
    vectorWeight?: number; // 向量权重 (默认 0.5)
    maxTokens?: number; // 上下文最大 token 数 (默认 8000)
  };
}
```

**响应体**：

```typescript
interface RAGQueryResponse {
  // 构建好的上下文
  context: {
    text: string; // 组装好的上下文文本
    sources: SourceReference[]; // 来源引用列表
    totalTokens: number; // 上下文总 token 数
  };

  // 详细检索结果
  chunks: ChunkResult[]; // 检索到的分块
  parentChunks: ParentChunkResult[]; // 父分块内容

  // 处理信息
  processingTime: {
    total: number; // 总耗时 (ms)
    search: number; // 检索耗时
    hyde?: number; // HyDE 耗时 (如启用)
    rerank?: number; // Rerank 耗时 (如启用)
  };
}

interface SourceReference {
  index: number; // 引用编号 [1], [2]
  documentId: string;
  documentTitle: string;
  pageStart?: number;
  pageEnd?: number;
  sectionTitle?: string;
  excerpt: string; // 摘要片段
  score: number; // 相关度分数
}
```
