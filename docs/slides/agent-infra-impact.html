Chatbot 模式下 KV Cache 在请求完成后立即丢弃，Agent 模式下必须在数分钟到数天的异步工作流中保持存活。GPU HBM 提供纳秒级延迟但容量固定不可扩展，系统 DRAM 带宽比 HBM 低一个数量级，导致 GPU 大量时间空转等待内存。

NVIDIA 在 CES 2026 推出 ICMSP（推理上下文内存存储平台），在 HBM 与通用存储间插入 G3.5 层——基于 Ethernet 闪存专为 KV Cache 优化。CXL 研究表明可减少 87% GPU 内存占用，同时维持亚 100ns 延迟。

"Agent AI 系统的出现正在将瓶颈从原始算力转移到内存容量、带宽和层级设计。定义性问题变成了 ‘AI 能多有效地记忆？’" — The Register, 2026.01.28

Chatbot 流量是用户驱动（白天高峰/夜间低谷），Agent 流量是任务驱动——全天候运行，可能一小时零请求、下一分钟突发 500 请求。OpenRouter 数据：2026.02 周处理 13万亿 tokens，比1月初 6.4万亿翻倍，与 OpenClaw 爆发直接相关。

Jensen Huang 称 OpenClaw 是"可能是最重要的一次软件发布"。H100 租赁价格自 2025.12 大幅反弹。基础设施响应：Lambda 部署千兆瓦级园区；Aethir 分布式网络覆盖 200+ 地点、44万 GPU 容器，定价比头部云厂商低 86%。NVIDIA 发布 NemoClaw 开源栈，一键部署全时 Agent。
