北美量产一代 网络：RDMA已是标配，但KV Cache流量暴露拥塞控制短板
核心信息（1分钟）

RDMA解决了"能搬"的问题。但KV Cache卸载流量是不规则大块突发，和训练allreduce在同一个fabric竞争，RoCEv2的拥塞控制在混合流量下性能急剧下降。

内容要点
  • KV Cache卸载 = 40GB在网络搬运，GPU空转等待
  • RoCEv2已是标配——GPU内存零拷贝直通远端存储
  • 两种流量特征对比：
  • allreduce：大块、规律、可预测、同步
  • KV Cache卸载：不规则突发、延迟极敏感、与allreduce竞争带宽
  • DCQCN为训练设计，面对混合流量拥塞时性能下降严重
  • 交换芯片缓存架构：shallow buffer vs deep buffer vs VOQ
  • SerDes信号完整性：持续高温下eye margin缩小，BER上升
  • Cisco Silicon One G300：102.4Tbps，网络利用率+33%，JCT-28%

内容要点

内容要点
