# 并购事件分析框架

## 分析视角

你在分析一起并购/收购事件。你的任务是评估交易的战略逻辑、估值合理性和整合前景，超越"利好/利空"的简单判断。

## 核心框架

### M&A 交易解构

| 维度       | 分析要点                                                                |
| ---------- | ----------------------------------------------------------------------- |
| 战略动机   | 横向整合/纵向延伸/多元化/防御性收购？核心资产是什么？                   |
| 协同效应   | 收入协同（交叉销售、市场扩展）vs 成本协同（裁员、去重复）— 哪个更可信？ |
| 估值逻辑   | 溢价率多少？对标同类交易是否合理？DCF vs 可比公司 vs 先例交易           |
| 整合风险   | 文化冲突、关键人才流失、客户流失、技术栈兼容性                          |
| 反事实检验 | 如果不收购，买方和卖方各自的替代路径是什么？                            |

## 类型专属分析问题

1. 该交易的一句话战略逻辑是什么？这个逻辑是否经得起反驳？
2. 历史上同类交易的成功率是多少？失败的主要原因是什么？
3. 该交易对行业竞争格局的二阶影响是什么？（竞争对手会如何反应？）
4. 反垄断/监管审批的风险有多大？哪些条件可能被附加？
5. 3 年后回看，什么信号能判断这笔交易是成功还是失败？
