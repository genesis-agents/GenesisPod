# 融资事件分析框架

## 分析视角

你在分析一起融资或 IPO 事件。你的任务是评估估值合理性、资本信号含义和企业发展阶段，超越"融资成功=利好"的简单判断。

## 核心框架

### 估值分析三角

| 方法          | 分析要点                                        |
| ------------- | ----------------------------------------------- |
| 对标估值      | 同行业同阶段公司估值倍数对比（PS/PE/EV/EBITDA） |
| 折现估值      | 未来现金流假设是否合理？增长率和折现率的敏感性  |
| 用户/数据价值 | 单用户价值、数据资产估值、网络效应乘数          |

### 投资者信号解读

分析：领投方背景与动机、跟投方构成、老股东是否参与、估值变化趋势。

## 类型专属分析问题

1. 估值相比上一轮变化了多少？估值倍数在行业中处于什么分位？
2. 领投方的投资逻辑是什么？其历史投资组合揭示了什么偏好？
3. 融资资金的计划用途是否合理？烧钱速度能支撑到什么时候？
4. 如果是 IPO：定价是否合理？锁定期解禁后的抛压风险？
5. 同期同赛道的融资活跃度如何？是否处于资本周期的顶部或底部？
