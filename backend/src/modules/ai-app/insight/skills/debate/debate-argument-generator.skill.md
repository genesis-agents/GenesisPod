# 辩论论点生成框架

## 论点构建方法论

### 1. 证据驱动论证

- 每个论点必须基于至少一条具体证据
- 引用格式：[来源标题](URL) 或脚注编号
- 优先使用定量数据、权威机构报告、同行评审研究

### 2. 论证结构

- **主张 (Claim)**：清晰的立场陈述（200-400字）
- **理由 (Reasoning)**：逻辑推理链条，连接证据与结论
- **证据 (Evidence)**：支撑材料，包含具体数据点
- **反驳准备 (Rebuttal prep)**：预判对方可能的反驳角度

### 3. 论点质量标准

- **逻辑严密性**：避免逻辑谬误（稻草人、滑坡、循环论证）
- **证据相关性**：证据直接支持主张，非旁证
- **说服力层次**：事实论据 > 专家意见 > 类比推理
- **新颖性**：在已有论点基础上推进讨论，避免重复

### 4. 置信度评估

- 0.9-1.0：有强有力的直接证据支持
- 0.7-0.9：有间接证据或多个弱证据交叉验证
- 0.5-0.7：基于合理推理但缺乏直接证据
- <0.5：高度推测性观点

### 5. 辩论策略

- **首轮**：建立核心论点，展示最强证据
- **中间轮**：回应对方论点，补充新证据
- **末轮**：总结核心优势，指出对方未能回应的关键问题
