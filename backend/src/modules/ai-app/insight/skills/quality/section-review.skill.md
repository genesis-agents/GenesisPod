# 章节质量审核 Skill

## 技能概述

审核研究 Agent 产出的章节内容质量，重点检测独立分析深度，防止纯证据拼接。

## 9 维评审体系

### 1. 内容完整性

- 是否覆盖了计划中的所有关键点 (`keyPoints`)
- 是否遗漏重要子主题

### 2. 分析深度

- 是否有因果推理和对比分析
- 是否超越了表面信息提供独到见解
- **核心指标**：独立分析内容占比

### 3. 证据使用

- 数据来源是否充分和可靠
- 证据是否恰当支撑论点

### 4. 逻辑连贯

- 段落间逻辑是否通顺
- 论证结构是否完整

### 5. 字数达标

- 是否达到计划的 `targetWords` 目标
- 字数是否分布均匀

### 6. 与章节计划的匹配度

- 是否紧扣 `description` 和 `keyPoints`
- 是否偏离了分配的研究范围

### 7. 写作质量

- 语言专业性和可读性
- 术语使用准确性

### 8. 图表使用

- 是否正确引用了分配的图表
- 图表引用格式是否规范

### 9. 与其他章节的衔接

- 是否与前序章节有适当过渡
- 是否避免了重复内容

## 扣分规则（5 条硬性规则）

| 规则         | 扣分 | 条件                                                     |
| ------------ | ---- | -------------------------------------------------------- |
| 无独立分析   | -20  | 纯证据转录/拼接，无因果推理、对比分析或独立判断          |
| 内容重复     | -15  | 与前序章节内容重复超过 30%                               |
| 套话开场     | -5   | 使用 "随着..."、"在当前..." 等模板化开头（每处扣一次）   |
| 模糊量词     | -5   | 使用 "大量"、"显著" 等模糊表述替代具体数据（每处扣一次） |
| 缺少因果分析 | -10  | 整个章节无因果链或对比分析                               |

## 独立分析深度评估

**关键判定标准**：章节必须包含独立分析（因果推理、对比分析、隐含洞察），不能只是证据转录。仅有"证据摘要拼接"的章节评分上限为 60 分。

5 个维度：

1. **因果推理** - 是否分析了现象背后的原因和影响
2. **对比分析** - 是否进行了有价值的横向或纵向对比
3. **隐含洞察** - 是否从数据中提炼出非显而易见的结论（须有严格推理支撑，不可杜撰）
4. **独立判断** - 是否有明确的立场和观点表达
5. **量化支撑** - 独立分析是否有数据佐证

## 审核决策

- `approved: true` + `score >= 60`: 通过
- `approved: false` + `revisionInstructions`: 需修改
- 最多修改 2 轮，之后强制通过（score >= 60 时）
