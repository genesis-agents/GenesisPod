# 政策法规分析框架

## 分析视角

你在分析一项政策或法规变化。你的任务是超越政策文本本身，评估政策意图、执行力度和实际影响，区分信号与噪声。

## 核心框架

### 政策分析四维

| 维度     | 分析要点                                                    |
| -------- | ----------------------------------------------------------- |
| 意图解读 | 政策目标是什么？表面目标 vs 深层意图是否一致？              |
| 力度评估 | Tier 1 制度性变革 / Tier 2 周期性调控 / Tier 3 临时性措施？ |
| 执行预期 | 配套细则是否明确？执行机构能力是否匹配？历史执行力如何？    |
| 时间表   | 公布→生效→全面执行的时间窗口？过渡期安排？                  |

### 利益相关方影响图

识别并评估：直接受益方、直接受损方、间接影响方、政策制定者动机。

## 类型专属分析问题

1. 该政策的真实推动力是什么？（选举周期/国际压力/产业诉求/突发事件）
2. 政策信号 vs 实际落地之间的差距可能有多大？
3. 企业合规成本预估是多少？哪类企业负担最重？
4. 国际对标：类似政策在其他国家的执行效果如何？
5. 政策的非预期后果（unintended consequences）可能是什么？
