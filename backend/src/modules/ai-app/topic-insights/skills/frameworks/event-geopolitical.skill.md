# 地缘政治分析框架

## 分析视角

你在分析一起地缘政治或贸易事件。你的任务是透过政治叙事理解经济逻辑，评估博弈均衡和供应链影响，避免情绪化判断。

## 核心框架

### 博弈论分析

| 要素       | 分析要点                                     |
| ---------- | -------------------------------------------- |
| 参与方     | 核心博弈方是谁？各方的核心利益和红线是什么？ |
| 策略空间   | 各方可选的行动方案及其成本收益               |
| 均衡分析   | 当前是否处于纳什均衡？有无帕累托改进空间？   |
| 承诺可信度 | 各方的威胁/承诺是否可信？有什么绑定机制？    |

### 供应链脆弱性评估

识别关键节点：单一来源依赖、替代方案成本与时间、库存缓冲能力、绕道路径。

## 类型专属分析问题

1. 各方的真实诉求是什么？公开立场与实际底线的差距？
2. 经济武器化程度：关税/制裁/出口管制的实际杀伤力有多大？
3. 联盟结构：哪些国家/组织会选边？中间地带有多大？
4. 供应链替代方案的成本和时间跨度是多少？
5. 历史类比：最接近的历史先例是什么？结局如何？
