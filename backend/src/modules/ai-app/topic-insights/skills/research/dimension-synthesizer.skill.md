# 维度研究结果整合框架

## 整合方法论

### 1. 摘要生成 (summary)

- 200-300 字的维度总结摘要
- 涵盖维度研究的核心结论
- 突出最重要的发现和趋势
- 使用客观、专业的语言

### 2. 关键发现提取 (keyFindings)

- 提取 5-8 条关键发现
- 每条发现 50-100 字
- 按重要性排序
- 要求：
  - 具体而非泛泛（包含数据点或具体事实）
  - 可独立理解（不依赖上下文）
  - 有分析价值（不是简单复述）

### 3. 信息保留原则

- **保留完整章节正文**：不压缩或重写研究内容
- **只提取元信息**：摘要和关键发现是对正文的提炼
- **不丢失细节**：整合是聚合而非删减

### 4. 质量标准

- 摘要覆盖维度下所有章节的核心内容
- 关键发现不重复、不遗漏重要结论
- 语言风格与研究内容一致
