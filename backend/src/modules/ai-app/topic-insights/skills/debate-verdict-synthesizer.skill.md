# 辩论终审综合框架

## 综合分析方法论

### 1. 辩论历程回溯

- 梳理各轮辩论的核心论点演变
- 识别关键转折点（某方提出决定性证据、某方承认弱点）
- 追踪未被回应的核心挑战

### 2. 最终裁定 (winningPosition)

- **pro**：正方整体更有说服力
- **con**：反方整体更有说服力
- **nuanced**：真相介于两者之间，需要细微区分

### 3. 关键论据提炼 (keyArguments)

- 从双方提取最具说服力的 3-5 个核心论据
- 按影响力排序
- 标注每个论据的证据强度

### 4. 遗留争议 (remainingContention)

- 双方均未能解决的核心分歧
- 需要更多数据才能判定的问题
- 价值观层面的根本性分歧（可能无法通过证据解决）

### 5. 综合观点 (synthesizedView)

- 超越正反二元对立，提出更完整的理解
- 整合双方有价值的洞察
- 提出条件化结论（在X条件下A成立，在Y条件下B成立）
- 给出实践建议和行动指引

## 综合原则

- 避免简单站队，追求真实洞察
- 承认复杂性，不过度简化
- 结论可操作，对决策者有指导价值
