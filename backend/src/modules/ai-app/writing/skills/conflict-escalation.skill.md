# 冲突升级与张力控制技能

## 核心原则

**永远不让角色舒服。** 每解决一个问题，必须引出一个更大的问题。

## 冲突层次金字塔

### 四层冲突（由内到外）

1. **内心冲突**：角色与自己的价值观、欲望、恐惧斗争
2. **人际冲突**：角色与其他角色的对抗（目标对立）
3. **社会冲突**：角色与体制、规则、传统的对抗
4. **命运冲突**：角色与不可控力量（自然、时代、命运）的对抗

### 规则

- 好的章节至少包含 **2层冲突** 同时运作
- 内心冲突是最深的驱动力——外部冲突必须触发内心冲突
- 反派的冲突也要合理——好的反派认为自己是正确的

## 张力升级曲线

### 章节内升级（3000字标准章节）

```
开头  ████░░░░░░  30% — 建立目标、暗示威胁
中段  ██████░░░░  60% — 冲突爆发、第一次挫败
转折  ████████░░  80% — 意外反转、计划失败
结尾  ██████████  100% — 悬崖/灾难/揭示
```

### 卷级升级

- 每卷末尾的危机必须**大于**卷开头
- 角色的赌注（stakes）必须持续增加：从个人安危→亲人安危→群体存亡
- 每卷至少一个"万劫不复"时刻

## 赌注升级技巧

| 赌注级别 | 具体内容     | 示例               |
| -------- | ------------ | ------------------ |
| 低       | 面子、小利益 | 比赛输赢           |
| 中       | 关系、地位   | 失去朋友/职位      |
| 高       | 自由、安全   | 被囚禁/追杀        |
| 极高     | 生命、所爱   | 至亲陷入危险       |
| 终极     | 信念、灵魂   | 必须做违背信念的事 |

## 禁止模式

- ❌ 冲突靠误会推动（"其实说开就没事了"）
- ❌ 外力解围（Deus ex machina）——突然来了个高人解决一切
- ❌ 角色实力碾压（没有挑战=没有故事）
- ❌ 连续3章没有冲突升级（读者会弃书）
