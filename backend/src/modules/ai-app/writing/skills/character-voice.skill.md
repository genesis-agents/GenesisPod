# 角色声音塑造 Skill

## 核心目标

让每个角色都有独特的"声音"，读者仅通过对话就能辨识说话者。

## 塑造维度

### 1. 语言习惯

- **用词选择**：教授用学术词汇，农民用土话
- **句式长度**：急性子说短句，思考者说长句
- **口头禅**：独特的词语或表达方式
- **语法特点**：方言、外语影响、语法错误

### 2. 说话方式

- **直接/委婉**：开门见山 vs 绕弯子
- **主动/被动**：主导对话 vs 跟随话题
- **情感表达**：外放型 vs 内敛型
- **幽默感**：有无、类型（讽刺、自嘲、冷笑话）

### 3. 知识背景

角色的对话应反映其：

- 教育程度
- 职业背景
- 人生阅历
- 兴趣爱好

### 4. 情感状态

同一角色在不同情绪下的表达差异：

- 紧张时语速加快、句子断裂
- 愤怒时用词激烈、音量提升
- 悲伤时语速放缓、停顿增多
- 兴奋时话多、跳跃性强

## 对话技巧

### 避免"对话头"

**差**：

```
"你好。"张三说。
"你好。"李四说。
"今天天气不错。"张三说。
```

**好**：

```
"你好。"张三推开门，带进一股寒气。
李四头也不抬："嗯。"
"今天天气不错。"见对方没反应，张三尴尬地找话说。
```

### 潜台词运用

人物真正想说的往往藏在话语之下：

- 用动作暗示真实情感
- 用沉默表达态度
- 用转移话题逃避问题
