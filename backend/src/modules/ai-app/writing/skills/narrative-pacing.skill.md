# 叙事节奏控制 Skill

## 节奏的本质

节奏 = 信息密度 × 时间跨度

- **快节奏**：高信息密度，短时间跨度
- **慢节奏**：低信息密度，长时间跨度或停留

## 加速技巧

当需要提升节奏时：

1. **缩短句子**
2. **增加动作描写**
3. **减少心理活动**
4. **使用跳跃式叙事**
5. **对话简短有力**

```
示例：
门撞开。枪响。他倒下。血蔓延开来。
```

## 减速技巧

当需要放慢节奏时：

1. **展开环境描写**
2. **深入内心世界**
3. **使用长句和复句**
4. **细致的感官描写**
5. **插入回忆或思考**

```
示例：
阳光从窗帘缝隙间漏进来，在地板上画出一道细长的光带。
他看着那道光，想起很多年前的那个下午...
```

## 张力曲线

一个章节的理想节奏曲线：

```
     /\
    /  \    /\
   /    \  /  \
  /      \/    \
 /              \
起 → 铺垫 → 高潮 → 回落 → 钩子
```

## 场景转换

### 硬切换

直接跳转，不做过渡：

```
***

三天后。
```

### 软切换

使用过渡语句：

```
等他再次睁开眼睛，已经是三天之后了。
```

### 交叉切换

两条线索交替推进，制造悬念。

## 节奏检查清单

- [ ] 连续的快节奏场景是否超过3个？
- [ ] 高潮场景前是否有足够铺垫？
- [ ] 情感高潮后是否有喘息空间？
- [ ] 章节结尾是否留有钩子？
