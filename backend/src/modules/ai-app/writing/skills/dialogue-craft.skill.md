# 对话写作进阶技能

## 核心原则：对话即冲突

每段对话至少要完成以下一项：推动情节、揭示角色、制造冲突。如果三项都没有，删掉它。

## 潜台词技巧（说A意思是B）

### 规则

- 角色**很少直接说出真实想法**
- 用间接方式表达：反问、讽刺、转移话题、沉默
- 读者应该能通过上下文理解真实含义

### 示例

- ❌ 直白："我很生气，因为你背叛了我。"
- ✅ 潜台词："茶凉了。" 她把杯子推到桌边。"你走吧。"

## 动作节拍（Action Beats）

### 规则

- 用角色的**微动作**替代"他说"/"她说"
- 动作要揭示情绪状态，不是填充空间
- 每3-4句对话插入一个动作节拍

### 常用动作节拍

| 情绪 | 动作                       |
| ---- | -------------------------- |
| 紧张 | 摩挲手指、咽口水、避开目光 |
| 愤怒 | 攥紧拳头、咬牙、摔东西     |
| 撒谎 | 摸鼻子、语速加快、眼神飘忽 |
| 压抑 | 深呼吸、低头、声音发颤     |

## 信息差对话

### 三层信息差

1. **角色A知道，角色B不知道** → 读者跟A视角时产生紧张
2. **读者知道，角色都不知道** → 反讽，读者为角色着急
3. **角色都知道，读者不知道** → 悬疑，对话暗示线索

## 禁止模式

- ❌ 两人你来我往平淡聊天（没有冲突目标）
- ❌ 长段独白式对话（超过5句无中断）
- ❌ 所有角色说话风格一样（千人一面）
- ❌ 用对话做信息倾倒（"正如你所知..."）
- ❌ 每句对话后都加"他笑着说"/"她温柔地说"（过度标注）
