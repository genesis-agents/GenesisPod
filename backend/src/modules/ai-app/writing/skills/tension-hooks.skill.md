# 悬念钩子与伏笔技能

## 章节开头钩子（前3段必须抓住读者）

### 六种钩子类型

1. **冲突对话式**：以争吵、质问、威胁的对话开场
2. **感官冲击式**：强烈的视觉/听觉/嗅觉/触觉描写
3. **极端困境式**：角色在危险、尴尬、绝望的处境中
4. **悬疑揭示式**：揭示一个令人震惊的事实或发现
5. **动作场景式**：角色正在奔跑、战斗、逃亡
6. **反转预期式**：与读者预期完全相反的情况

### 禁止的开头

- ❌ "在一个XX的世界里..."（世界观介绍）
- ❌ 角色醒来描写日常（除非醒来就面临危险）
- ❌ 大段环境描写没有角色行动
- ❌ 回忆前情提要

## 章节结尾悬念（让读者必须翻到下一章）

### 五种结尾悬念

1. **悬崖式**：在最紧张的瞬间戛然而止
2. **揭示式**：最后一句揭示关键信息
3. **威胁式**：新的更大威胁出现
4. **反转式**：之前认为安全的事物变成危险
5. **选择式**：角色面临两难选择，读者不知道结果

### 禁止的结尾

- ❌ 角色"暗下决心"、"心中坚定"
- ❌ "她知道，前方的路..."展望式
- ❌ "这只是开始"预告式
- ❌ 内心独白总结本章内容
- ❌ 主题升华和人生感悟

## 伏笔系统

### 伏笔类型

| 类型     | 埋设     | 回收间隔 | 示例                 |
| -------- | -------- | -------- | -------------------- |
| 短线伏笔 | 细节描写 | 1-3章    | 角色注意到的异常细节 |
| 中线伏笔 | 事件暗示 | 5-10章   | 某个角色的可疑行为   |
| 长线伏笔 | 主题铺垫 | 全书     | 开篇出现的象征物     |

### 伏笔规则

1. 埋设时要**自然不突兀**——融入正常叙事中
2. 每5章至少回收一个短线伏笔——给读者"原来如此"的满足感
3. 长线伏笔在高潮段回收——产生最大震撼
4. 禁止"遗忘伏笔"——埋了必须收
